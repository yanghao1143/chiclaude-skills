## 技能简介

ClawDirect 工具，用于简化命令行操作和自动化。

**适用场景**：命令行自动化、脚本执行、工作流简化。

## 核心功能

### 命令简化
- 别名管理
- 命令组合
- 参数预设

### 工作流自动化
- 任务编排
- 条件执行
- 错误处理

---

## 使用方式

### 基本命令
```bash
clawdirect run <command>
clawdirect alias <name> <command>
clawdirect list
```
