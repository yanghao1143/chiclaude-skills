## 技能简介

创建设计文档的 Markdown 模板和指南。

**适用场景**：编写设计文档、技术规范、架构文档。

---

## 设计文档结构

### 概述
- 背景和动机
- 目标和非目标
- 成功指标

### 设计
- 高层架构
- 组件设计
- 数据模型
- API 设计

### 实现计划
- 阶段划分
- 依赖关系
- 风险缓解

### 替代方案
- 考虑过的其他方案
- 为什么没选择
