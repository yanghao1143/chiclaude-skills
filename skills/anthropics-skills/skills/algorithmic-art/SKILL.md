## 技能简介

使用 p5.js 创建算法艺术，支持随机种子和交互式参数探索。

**适用场景**：用户请求使用代码创建艺术、生成艺术、算法艺术、流场或粒子系统。创建原创算法艺术而不是复制现有艺术家的作品以避免版权侵犯。

## 核心概念

### 算法哲学创建
算法哲学是通过代码表达的计算美学运动。输出：
- .md 文件（哲学）
- .html 文件（交互式查看器）
- .js 文件（生成算法）

### 两步流程
1. **算法哲学创建**（.md 文件）
2. **通过 p5.js 生成艺术表达**（.html + .js 文件）

## 关键原理

### 种子随机性（Art Blocks 模式）
```javascript
// 始终使用种子以确保可重现性
let seed = 12345;
randomSeed(seed);
noiseSeed(seed);
```

### 参数结构
```javascript
let params = {
  seed: 12345,  // 始终包含种子
  // 颜色
  // 添加控制算法的参数：
  // - 数量（多少？）
  // - 比例（多大？多快？）
  // - 概率（多可能？）
  // - 比率（什么比例？）
  // - 角度（什么方向？）
  // - 阈值（行为何时改变？）
};
```
